My Projects

Path Finding Visualizer

This project is based on Dijkstra's algorithm for finding the shortest possible path between a starting node and a target node using a Python module called PyGame. The starting and target nodes, along with the blocking nodes or walls, can be set by the user and the shortest path between them, ignoring all the intermediate walls, will be auto-generated by the program.

RUN MACHINE - cricket app

This project is an android cricket application developed using Flutter with the help of APIs. The app offers real-time cricket score info, results, latest cricket match fixtures, trending cricket news, current ICC men and women player rankings, current ICC men and women team rankings, IPL leaderboard and player statistics and also provides fantasy cricket tips for the users.

Cornflix - movies app

This project is an android movies application developed using flutter with the help of TMDB and other APIs. This app shows the details of latest, top-rated and upcoming movies, TV shows and anime. This app is the perfect way to know about movies, TV shows, anime and stay up-to-date on the latest releases, popular and top-rated contents.
